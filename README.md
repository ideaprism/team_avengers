## 🚀 사용 방법

이 저장소는 **Antigravity (AI Agent)**와 연동하여 언제 어디서든 동일한 페르소나 환경을 제공합니다.

1. **GitHub 연동**: 이 저장소를 Clone하거나 다운로드하여 사용합니다.
2. **에이전트 호출**: 각 페르소나의 `.md` 파일을 Antigravity에게 읽혀 해당 인격을 소환합니다.
3. **이미지 자산**: 모든 이미지는 `raw.githubusercontent.com`을 통해 제공되므로, 외부 환경에서도 깨짐 없이 표시됩니다.
